# Live Τηλεματική ΟΑΣΑ

## Προϋπόθεση
Απαιτείται γραπτή άδεια χρήσης δεδομένων ΟΑΣΑ.

## Πώς τρέχει
1. Ανέβασε τον φάκελο σε HTTPS server
2. Άνοιξε index.html
3. Διάλεξε γραμμή → κατεύθυνση → στάση

## API endpoints
- webGetLines
- getRoutesForLine
- getStopsForRoute
- getStopArrivals
- getBusLocation

## Ασφάλεια
- Χρήση backend proxy συνιστάται για production
- Rate limiting & caching
